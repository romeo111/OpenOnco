Clone-platform identity matters because the Blueprint PD-L1 Comparison phase 1 + 2 studies (Hirsch JTO 2017; Tsao JTO 2018) showed 22C3 / 28-8 / SP263 as analytically near- equivalent for tumor-cell staining (NSCLC TPS), but SP142 (Ventana, atezolizumab CDx) markedly under-stains tumor cells and is NOT interchangeable. CPS (combined positive score; tumor + lymphocyte + macrophage staining) is operationally clone-22C3-anchored across pembrolizumab indications — substituting SP263 / 28-8 / SP142 for CPS-driven indication selection is off-label.
