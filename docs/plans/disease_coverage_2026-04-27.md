## Метрики

- **#Bio** — distinct biomarkers, на які посилаються Indications + Regimens цієї хвороби
- **#Drug** — distinct drugs у regimens цієї хвороби
- **#Ind / #Reg / #RF** — Indications / Regimens / RedFlags для цієї хвороби
- **1L / 2L** — наявність Algorithm для першої / другої+ лінії
- **Quest** — `✓` hand-authored questionnaire / `STUB` auto-generated / `—` відсутній
- **Fill%** = composite з 8 ентити-типів: ≥1 indication, ≥1 regimen, ≥1 biomarker, ≥1 drug, ≥1 redflag, 1L algo, questionnaire, workup
- **Ver%** = % indications цієї хвороби з `reviewer_signoffs ≥ 2` (CHARTER §6.1)

> **Verified взагалі = 0** для всіх — діє dev-mode signoff exemption (`project_charter_dev_mode_exemptions`)
> до призначення Clinical Co-Leads. Це навмисно, не bug.
