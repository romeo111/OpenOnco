# Clinical Signoff Packet Template

Use this template for review batches created from audit remediation work.
One packet should be small enough for a reviewer to approve, reject, or
return with questions without rediscovering the entire evidence trail.

## Reviewer Checklist

- [ ] Entity IDs and disease context are correct.
- [ ] Source refs resolve and support the changed field.
- [ ] No outcome, dose, label, reimbursement, or safety claim is inferred
      beyond the source.
- [ ] Unknown/deferred fields are explicitly marked.
- [ ] Patient-safety or access implications are captured.
- [ ] The packet is ready for second reviewer or final signoff.
