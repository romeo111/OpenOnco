Starting the OpenOnco engine

The first launch takes ~10–20 seconds — the engine loads directly into your browser. Patient data never leaves your device.

  1. Loading the in-browser runtime (~6 MB)
  2. Setting up the environment
  3. Loading the OpenOnco knowledge base
  4. Verifying the clinical KB
  5. Building a personalised plan

If it's working now, next time it'll be ~5 s — the engine stays in memory.
