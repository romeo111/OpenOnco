Нотатки

Distinct from BIO-FGFR2 (gene-level alterations: fusion / amplification / mutation). FGFR2b-IHC is the protein-level companion biomarker for bemarituzumab — anti-FGFR2b chimeric IgG1 mAb with afucosylated Fc enhancing ADCC. FIGHT phase 2 (Wainberg, Lancet Oncol 2022) used FGFR2b 2+/3+ in ≥10% of tumor cells as eligibility threshold; phase 3 FORTITUDE-101 (cite as-is — source stub pending citation upgrade) uses the same threshold. Companion-diagnostic IVD development ongoing alongside FORTITUDE-101 readout. Note: FGFR2-pan IHC and FGFR2b-IHC are NOT interchangeable — only the IIIb-isoform-selective antibody enables bemarituzumab patient selection.
