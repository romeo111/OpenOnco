Нотатки

Companion-diagnostic clone-specific entity. Distinct from BIO-PDL1-TPS / BIO-PDL1-CPS (scoring conventions) because the assay clone matters: pembrolizumab indications require 22C3 specifically per FDA labeling, and 22C3 vs SP142 are NOT interchangeable on immune-cell-weighted scoring. Used by render layer to surface assay-validity warnings when a patient profile reports PD-L1 testing with a non-22C3 clone for a pembrolizumab-routed indication.
