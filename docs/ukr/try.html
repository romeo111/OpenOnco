Готую двигун OpenOnco

Перший запуск триває ~10–20 секунд — двигун завантажується безпосередньо у твій браузер. Дані пацієнта не лишають твого пристрою.

  1. Готую обчислювач у браузері (~6 МБ)
  2. Налаштовую середовище
  3. Завантажую базу знань OpenOnco
  4. Звіряю клінічну базу
  5. Будую персональний план

Якщо зараз вийшло, наступного разу буде ~5 с — двигун залишається в пам'яті.
